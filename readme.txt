== Description ==

Add tags to pages and display them in a tagcloud widget.

== Installation ==

1. Upload `page-tag-cloud` to the `/wp-content/plugins/` directory
2. Activate the plugin through the 'Plugins' menu in WordPress

== Frequently asked questions ==

= Does this plugin support a tag cloud on my homepage which displays my latest posts ? =

It does. Select the 'Show on home' option of tags you would like to show on your latest posts homepage.
